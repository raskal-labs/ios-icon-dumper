a unified kit of "tools" for grabbin iOS app icons that may or may not work for you - better probably exists elsewhere, i promise...

**Release:** v2.0.0
**Compatibility:** iOS 15+ (Rootless/Rootful), macOS, Linux

## üöÄ Usage

### 1. Discovery & Extraction (Run `Extractor` on Phone)
Use the Extractor (v2) to scan your device, build the master ID list, and dump your local icons.

```bash
# Discover apps & Extract local icons
./ios-icon-extractor.sh discover
./ios-icon-extractor.sh run
Output: artifacts/extracted/

2. High-Res Enrichment (Run Downloader Anywhere)
Use the Downloader (v1) to read the list generated by the Extractor and fetch higher quality artwork.

Bash

# Download 512px icons for the discovered IDs
./ios-icon-downloader.sh run
Output: artifacts/downloaded/
